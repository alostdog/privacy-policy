# Privacy Policy Pages

隐私政策页面展示项目

## 📄 内容

- `index.html` - 隐私政策网页版本
- `privacy-policy.md` - 隐私政策 Markdown 版本

## 🚀 使用方式

### GitHub Pages 部署

1. 进入仓库 Settings → Pages
2. Source 选择 `Deploy from a branch`
3. Branch 选择 `main`，文件夹选择 `/ (root)`
4. 保存后访问 `https://alostdog.github.io/privacy-policy/`

### 自定义内容

编辑 `privacy-policy.md` 或 `index.html` 文件，根据你的实际需求修改隐私政策内容。
